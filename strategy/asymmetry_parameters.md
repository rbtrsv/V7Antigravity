# V7 Capital — Parametri de Analiză | Mapare pe Asimetrie

> **Model:** `Câștig investiție = P(win/lose) × Expected Return × Alocare`
>
> Fiecare parametru de mai jos influențează fie **probabilitatea de a face/pierde bani (P)**, fie **randamentul așteptat al investiției / pay-off (R)**, fie ambele.

---

## PILON 2: COMPANIE / MODEL DE BUSINESS

| # | Parametru | Influențează P | Influențează R | Logica mapării |
|---|-----------|:--------------:|:--------------:|----------------|
| C1 | **Product value proposition** | ✅ | | Rezolvă o problemă reală și acută? Cu cât mai acută nevoia, cu atât mai probabil că piața cumpără |
| C2 | **Diferențiator sustenabil / Moat** | ✅ | ✅ | Reduce competiția (P) + oferă pricing power și valoare la exit (R). Include: high switching costs (data lock-in, integrations), proprietary data/IP (patents, unique dataset), brand/trust (regulated industry), scale economies (marketplace, infrastructure) |
| C3 | **Go-to-market (achiziție + retenție clienți)** | ✅ | | Strategie credibilă și scalabilă de a ajunge la clienți = reduce riscul de a nu vinde |
| C4 | **Monetizare, Pricing & Recurență** | ✅ | ✅ | Model de revenue, pricing power, venituri recurente (subscriptions, contracts) = predictibilitate (P) + determină magnitudinea return-ului (R). Include retenția veniturilor |
| C5 | **Unit economics** | ✅ | ✅ | Sănătoase = mai puțin risc de a arde cash (P) + marje bune = return mai mare (R) |
| C6 | **Scalabilitate (profitabilitate marginală)** | | ✅ | Efectul creșterii în profitabilitatea marginală determină cât de mare poate fi return-ul |
| C7 | **Margin of Safety** | ✅ | | **Se aplică diferit pe tip de investiție.** La **PE & Public Markets:** marje mari, debt zero/mic, active tangibile, cash flow = airbag intrinsec. La **VC / Early Stage:** MoS clasic nu se aplică (pre-revenue, no assets) — MoS se creează prin **Deal Structuring** (liquidation preference 1x, anti-dilution, ratchet — vezi D1-D11) |
| C8 | **Structura portofoliului de clienți** | ✅ | | Concentrare mare = risc mare; dispersie = stabilitate |
| C9 | **Opțiuni de pivotare** | ✅ | | Dacă planul A nu merge, există B și C? Versatilitate = reduce riscul de pierdere totală |
| C10 | **Potențialul de a crește piața adresabilă** | | ✅ | Business-ul poate expanda TAM, nu doar lua cotă → upside disproporționat |
| C11 | **Evaluarea la intrare (entry valuation)** | | ✅ | Cu cât mai mică evaluarea vs. potențial, cu atât mai mare pay-off-ul la exit |
| C12 | **Viteza de creștere** | | ✅ | Creștere rapidă = compoundare mai puternică = return mai mare, mai repede |
| C13 | **Nevoia de capital / Cash Burn** | ✅ | ✅ | Cash conversion cycle scurt + puțin capital necesar = mai puțin risc de lichiditate (P) + mai puțin dilution (R). Include runway |
| C14 | **Maturitatea produsului vs. piață** | ✅ | | Product-market fit dovedit = risc redus. Dacă e pre-PMF = risc mai mare |
| C15 | **Capacitatea de a atrage finanțare** | ✅ | ✅ | Abilitatea de a atrage capital suplimentar reduce riscul de a rămâne fără cash (P) + permite accelerarea creșterii (R) |
| C16 | **Client feedback** | ✅ | | Feedback pozitiv și validare din piață = confirmare că produsul rezolvă o nevoie reală, reduce riscul de execuție |

## PILON 3: INDUSTRIA

| # | Parametru | Influențează P | Influențează R | Logica mapării |
|---|-----------|:--------------:|:--------------:|----------------|
| I1 | **Dimensiunea pieței (TAM / SAM / SOM)** | | ✅ | Piață mare = ceiling de creștere (și implicit de return) mai mare |
| I2 | **Viteza de creștere a industriei** | | ✅ | Industrie în creștere = tailwind natural → return mai mare |
| I3 | **Nivel competitiv, Concentrare & Maturitate** | ✅ | | Piață fragmentată = mai greu de câștigat; piață cu bariere = mai ușor de apărat poziția. Include riscul de timing — un produs prea avangardist (ahead of market) poate eșua din cauza lipsei de product-market fit |
| I4 | **Trenduri macro favorabile** | | ✅ | Tailwinds (demografice, tech, regulatorii) amplifică potențialul de return |
| I5 | **Existența buyer-ilor strategici / Opțiuni de exit** | | ✅ | Piață cu acquireri activi = exit mai probabil, la multipli mai buni |
| I6 | **Dinamică locală RO (lag vs Vest)** | | ✅ | România în urmă = convergence play → potențial de creștere accelerată |
| I7 | **Riscul de reglementare** | ✅ | | Industrii intens reglementate = risc de schimbări legislative sau de conformitate care pot afecta modelul de business |
| I8 | **Rezistență la cicluri economice** | ✅ | | Industrii defensive = probabilitate mai mare de a supraviețui recesiuni |
| I9 | **Efecte de rețea / Winner-takes-most** | ✅ | ✅ | Dacă industria tinde spre monopol natural, jucătorul corect câștigă masiv (R), dar și riscul de a pierde tot crește dacă nu ești câștigătorul (P) |

## PILON 4: DEAL STRUCTURING

| # | Parametru | Influențează P | Influențează R | Logica mapării |
|---|-----------|:--------------:|:--------------:|----------------|
| D1 | **Liquidation Preference** | ✅ | | Prioritate la distribuirea capitalului în caz de exit sau lichidare — protejează downside-ul, asigură recuperarea capitalului înaintea fondatorilor |
| D2 | **Anti-dilution Protection** | ✅ | ✅ | Protejează contra diluției în down rounds (P) + păstrează ownership-ul care determină magnitudinea return-ului (R) |
| D3 | **Drag-Along Rights** | | ✅ | Permite forțarea unui exit chiar dacă minoritarii se opun — deblochează lichiditate și accelerează realizarea return-ului |
| D4 | **Tag-Along Rights** | ✅ | | Protejează investitorul minoritar — dacă fondatorii/majoritarii vând, avem dreptul să vindem în aceleași condiții. Evită scenariul de a rămâne captiv |
| D5 | **Earnout / Ratchet Provisions** | ✅ | ✅ | De-riscă evaluarea la intrare (P) — plătim mai mult doar dacă performanța se confirmă. Aliniază incentivele pentru creștere (R) |
| D6 | **Board Seat / Governance Rights** | ✅ | | Influență directă pe decizii strategice — early warning system + capacitatea de a corecta direcția înainte de a fi prea târziu |
| D7 | **Protective Provisions / Veto Rights** | ✅ | | Dreptul de a bloca acțiuni distructive: over-dilution, vânzare active sub valoare, debt excesiv, schimbarea naturii business-ului |
| D8 | **Vesting & Lock-up fondatori** | ✅ | | Asigură că fondatorii rămân pe termen lung — reduce flight risk și menține continuitatea execuției |
| D9 | **Pro-rata / Pre-emption Rights** | | ✅ | Dreptul de a menține sau crește participația în runde viitoare la termeni favorabili — maximizează upside-ul pe investițiile câștigătoare |
| D10 | **Information Rights** | ✅ | | Raportare financiară periodică, acces la audit, KPI dashboard — detectarea timpurie a problemelor reduce riscul de surprize negative |
| D11 | **Exit Clauses / Put Options** | ✅ | ✅ | Mecanism garantat de ieșire = evită blocajul investiției (P) + setează un floor pe return (R). Include buy-back clauses |

### Observații din mapare:

1. **Fondatorii** influențează **masiv P(win/lose)** — aproape toți parametrii sunt pe coloana de probabilitate. Calitatea fondatorilor determină execuția, iar execuția determină dacă business-ul supraviețuiește.

2. **Compania** e un **mix** — unii parametri pe P (safety net, diversificare risc), alții pe R (scalabilitate, pricing, evaluare intrare), câțiva pe ambele (diferențiator, unit economics, monetizare).

3. **Industria** influențează aproape exclusiv **Expected Return** — dimensiunea pieței, trenduri, opțiuni de exit. Parametrii pe P: competiția/maturitate, reglementare, rezistență la cicluri.

4. **Deal Structuring** influențează **predominant P** — majoritatea proviziilor contractuale protejează investitorul contra downside-ului. Excepții pe R: drag-along (deblochează exit), pro-rata (maximizează upside), anti-dilution și earnout (ambele).

5. **Asimetria se naște la intersecție:** Cele mai puternice semnale de asimetrie pozitivă apar când ai fondatori puternici (P mare) într-o industrie mare cu trenduri favorabile (R mare), cu un business model cu unit economics sănătoase (P + R), la o evaluare rezonabilă (R mare), protejat de o structură de deal care limitează downside-ul (P) și amplifică upside-ul (R).

6. **Margin of Safety funcționează diferit pe tipuri de investiții:**
   - **PE & Public Markets:** MoS = intrinsec. Marje, active tangibile, cash flow, debt scăzut. Întrebarea: „Dacă merge rău, ce recuperăm din business-ul în sine?"
   - **VC / Early Stage:** MoS intrinsec nu există (pre-revenue, no assets). MoS se creează prin **Deal Structuring**: liquidation preference 1x (recuperăm investiția înaintea fondatorilor), anti-dilution (ne protejăm ownership-ul), ratchet (plătim mai mult doar dacă performanța se confirmă). Întrebarea: „Ce clauze ne protejează downside-ul?"
