# V7 Capital - Governance

**Sursă:** SHA (Jan 2026)

---

## Structură Generală

```
┌─────────────────────────────┐
│   GMS (General Meeting)     │  ← Toate deciziile majore (75%)
│   Toți Shareholders         │
└─────────────┬───────────────┘
              │
┌─────────────▼───────────────┐
│   Supervisory Board (5)     │  ← Strategie + aprobare investiții >10%
│   4 × ≥3% + 1 × <3%         │
└─────────────┬───────────────┘
              │
┌─────────────▼───────────────┐
│   Management Board (3)      │  ← Operațiuni zilnice
│   GM + 2 Managers           │
└─────────────────────────────┘
```

## 1. General Meeting of Shareholders (GMS)

### 1.1 Vot
- **1 acțiune = 1 vot**
- Decizii normale: majoritate conform Companies Law

### 1.2 Decizii cu majoritate 75%

Următoarele necesită **prezența și votul favorabil al 75% din acțiuni:**

| # | Decizie |
|---|---------|
| i | Schimbarea tipului companiei de la evergreen |
| ii | Merger, spin-off sau alte reorganizări |
| iii | Lichidare și dizolvare |
| iv | Schimbare structură management |
| v | Procedură insolvență |
| vi | Numire/înlocuire GM (GM-ul se abține) |
| vii | Investiții în sectoare interzise |
| viii | Modificare remunerație EMT (>10%) |
| ix | Schimbare reguli redemption |
| x | Modificare materială Investment Strategy |
| xi | Stabilire clase de acțiuni |
| xii | Modificări prejudiciale în Articles of Association |

**"Modificare materială" înseamnă DOAR:**
- Adăugare/eliminare categorii investiții
- Modificare listă investiții interzise
- Creștere limite debt cu >25%

### 1.3 Conflict de interese
- Declarare obligatorie
- Abținere de la vot
- Voturile în conflict se exclud din quorum

## 2. Supervisory Board (BoA)

### 2.1 Componență
| Poziție | Număr | Nominator |
|---------|-------|-----------|
| Membri | 4 | Shareholders ≥3% |
| Membri | 1 | Shareholders <3% |
| **Total** | **5** | |

### 2.2 Numire
- **Vot cumulativ:** fiecare shareholder are voturi = nr. acțiuni × poziții disponibile
- **Mandat:** 4 ani, reînnoire nelimitată
- **Revocare:** de către GMS

### 2.3 Chairman
- Ales de membrii Supervisory Board
- Prerogative conform Companies Law

### 2.4 Remunerație
- **Nicio remunerație** pentru poziție
- Doar rambursare cheltuieli documentate

### 2.5 Consultanți externi
- Max **0.3% NAV/an** (fără aprobare GMS)

### 2.6 Ședințe
- Min. **1× pe trimestru**
- Convocare: email, 7 zile înainte
- Materiale: 3 zile înainte
- **Quorum:** >50% membri + Chairman prezent
- Posibil: remote, circulație

### 2.7 Voting
| Tip decizie | Voturi necesare |
|-------------|-----------------|
| Normal | **4 din 5** voturi favorabile |
| >15% Assets sau modificare Strategy | **Unanimitate** |
| Rejectat de 2× sau fără quorum de 3× | Trece la GMS |

### 2.8 Atribuții Exclusive (necesită aprobare Supervisory Board)

| # | Threshold | Decizie |
|---|-----------|---------|
| i | >10% Assets | Investiție sau exit |
| ii | >10% Assets | Debt la nivel Company |
| iii | >2 ani | Extindere perioadă investiție |
| iv | - | Excepții de la Valuation Methodology |
| v | - | Investiții în afara Investment Strategy |
| vi | - | Investiții fără exit rights |
| vii | >5% Assets | Buy-backs sau dividend distributions |
| viii | >3× EBITDA | Debt la nivel Portfolio Project |
| ix | - | Schimbare Investment Strategy |
| x | - | Conflicte de interese |
| xi | - | Excepții KYC |
| xii | >0.3% NAV | Costuri individuale sau serie de costuri |
| xiii | - | SOP Award: eligibilitate, alocare, excepții |

## 3. Management Board

### 3.1 Componență
- **3 membri** (inclusiv GM)
- Specializare: 1-2 PE, 1 Capital Markets
- **Mandat:** 4 ani, reînnoire nelimitată
- Numire: de către Supervisory Board
- GM: numit de GMS (75%)

### 3.2 Reprezentare
- **GM singur** reprezintă compania

### 3.3 Ședințe
- Min. **1× pe lună**
- Convocare: 2 zile înainte
- Quorum: 2 membri + President

### 3.4 Authority Levels pentru Investiții

| Exposure | Decision Maker | Condiții |
|----------|----------------|----------|
| ≤5% Assets | GM singur | - |
| 5-10% Assets | Management Board | GM vot favorabil + 0 voturi negative |
| >10% Assets | Supervisory Board | - |
| >15% Assets | Supervisory Board unanim | - |

---

## 4. Conflict of Interest Rules

**Aplicabile la toate nivelurile:**

1. **Declarare** obligatorie a potențialelor conflicte
2. **Excludere** din discuții și vot pe subiectul în conflict
3. **Votul conflictual** se exclude din quorum și majoritate
4. **Chairman în conflict:** 2 alți membri pot decide excluderea
5. **Confidențialitate:** membrul conflictual nu primește info
